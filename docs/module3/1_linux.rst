Some VIM features I used for this project

::

    Shift + G           { last line }
    Shift + $           { end of line }
    Shift + V           { block editor }
    y                   { yank }
    Shift + P           { paste }
    dd                  { delete/cut }

To edit files in VIM with tabs use the command  ``:tabedit <filename>``

 * gt
 * gT

    :earlier 2m
